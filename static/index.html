Arquitetura

🏛️
Clean Architecture
Domain → Application → Infrastructure → Interface. Dependências sempre apontam para dentro.
⚡
Event-Driven
Eventos de domínio desacoplam módulos. RabbitMQ como broker de mensagens assíncronas.
🔒
JWT + Roles
Autenticação stateless. Roles: PATIENT, DOCTOR, ADMIN com controle por rota.
🐳
Docker Compose
API + PostgreSQL 16 + RabbitMQ 3.13 orquestrados com health checks automáticos.
📦
Repository Pattern
Portas e adaptadores isolam a persistência. Fácil trocar ORM ou banco sem alterar casos de uso.

Eventos de Domínio (RabbitMQ)

🔬 ExamPerformed → exam.performed Disparado ao criar um exame
📋 ReportIssued → report.issued Disparado ao emitir um laudo
✅ ResultAvailable → result.available Notifica disponibilidade do resultado
